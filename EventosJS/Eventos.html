Eventos en javascript

Uso del evento "oninput"

Ejecuta un script cuando el usuario ha ingresado algo en la caja de texto

Escribe algo en el campo de texto para activar una función.


Uso del evento "onpaste"

Este evento sirve para pegar un texto después de ser cortado
